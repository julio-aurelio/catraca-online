## ✨ Funcionalidades Principais

- 🎫 **Validação de acesso** - Simula a liberação ou bloqueio de entrada
- ✨ **Animações interativas** - Feedback visual durante a validação de dados
- 🏢 **Identidade visual** - Logo da academia integrada ao sistema
- 📱 **Design responsivo** - Funciona em diferentes tamanhos de tela
- ⚡ **Interface intuitiva** - Foco na simplicidade e usabilidade
